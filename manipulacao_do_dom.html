Acesso a elementos

O acesso a elementos do DOM em JavaScript pode ser feito de várias maneiras, dependendo do requisito específico. Aqui estão algumas das principais maneiras de acessar elementos do DOM:

getElementById
Esse método é usado para obter uma referência a um elemento pelo seu ID único. Ele retorna o elemento correspondente como um objeto.

getElementsByClassName
Esse método é usado para obter uma referência a um elemento pelo seu ID único. Ele retorna o elemento correspondente como um objeto.

getElementsByTagName
: Esse método é usado para obter uma lista de elementos que têm a mesma tag HTML. Ele retorna uma coleção de elementos correspondentes.
