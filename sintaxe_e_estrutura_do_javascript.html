Variáveis

Uma variável em programação é um nome simbólico que representa um local de armazenamento para dados. Ela permite que você armazene, acesse e manipule valores durante a execução de um programa.

Em JavaScript, existem três palavras-chave principais para declarar variáveis: var, let e const. Vamos explorar cada uma delas e fornecer exemplos de uso:

var

A palavra-chave var foi a forma tradicional de declarar variáveis em JavaScript. Ela tem escopo de função ou, se declarada fora de uma função, escopo global. Isso significa que uma variável declarada com var é acessível em toda a função em que foi declarada ou, se for global, em todo o programa. No entanto, o escopo de var não é limitado a um bloco de código específico. O var sofre elevação (hoisting), o que significa que a declaração da variável é movida para o topo do seu escopo antes da execução do código.

let

A palavra-chave let é usada para declarar variáveis com escopo de bloco em JavaScript. Com o let, você pode criar variáveis que são acessíveis apenas dentro do bloco em que foram declaradas. Essas variáveis não são elevadas (hoisted) para o topo do escopo e não podem ser redeclaradas no mesmo escopo. O let é útil em loops for para evitar problemas de escopo.

const

A palavra-chave const também foi introduzida no ES6 e é usada para declarar constantes em JavaScript. Uma constante é uma variável com um valor imutável, ou seja, seu valor não pode ser alterado após a sua atribuição inicial. As constantes também têm escopo de bloco e não sofrem elevação (hoisting). Elas devem ser declaradas e inicializadas ao mesmo tempo, e não podem ser reatribuídas a outro valor posteriormente.

Em resumo, var é a forma mais antiga de declarar variáveis em JavaScript com escopo de função ou global, let é usada para declarar variáveis com escopo de bloco e const é usada para declarar constantes com valor imutável. Cada uma dessas palavras-chave tem diferentes comportamentos em relação ao escopo e à reatribuição de valores.

Tipos de dados

Tipos de dados são categorias que definem o tipo de valor que uma variável ou uma expressão pode armazenar ou manipular em um programa. Os tipos de dados ajudam a determinar o comportamento e as operações que podem ser realizadas com os valores.

Em JavaScript, alguns tipos de dados comuns incluem:

Números (Number)

Representam valores numéricos, como inteiros e números de ponto flutuante (decimais). Por exemplo: 5, 3.14, -10.

Strings

Representam sequências de caracteres. São definidas entre aspas simples ou duplas. Por exemplo: "Olá", 'Mundo', "123".

Booleanos (Boolean)

Representam os valores lógicos verdadeiro (true) e falso (false). São frequentemente usados em expressões condicionais. Por exemplo: true, false.

Arrays

Representam coleções ordenadas de valores, que podem ser de qualquer tipo de dados. São definidos usando colchetes [] e os elementos são separados por vírgulas. Por exemplo: [1, 2, 3], ['a', 'b', 'c'].

Objetos (Object)

São estruturas de dados que permitem agrupar propriedades e métodos relacionados em um único objeto. As propriedades são definidas como pares chave-valor, separados por vírgulas e delimitados por chaves {}. Por exemplo: { nome: 'João', idade: 30 }.

Null

Representa a ausência intencional de qualquer valor ou objeto. É um valor especial que indica a ausência de um valor válido. Por exemplo: null.

Undefined

Indica que uma variável foi declarada, mas ainda não foi atribuído nenhum valor a ela. É o valor padrão para variáveis não inicializadas. Por exemplo: undefined.

Operadores

Os operadores em JavaScript são símbolos especiais que executam operações em valores e retornam um resultado. Eles são usados para realizar cálculos matemáticos, comparações, atribuições, lógica booleana e muito mais.

Existem vários tipos de operadores em JavaScript:

Operadores Aritméticos:
Realizam operações matemáticas, como adição (+), subtração (-), multiplicação (*), divisão (/) e resto da divisão (%).

Operadores de Atribuição:
Atribuem um valor a uma variável, utilizando o símbolo de igual (=), e também podem ser combinados com operadores aritméticos para realizar operações e atribuir o resultado à variável (ex.: +=, -=, *=, /=).

Operadores de Comparação:
Compararam dois valores e retornam um valor booleano (verdadeiro ou falso). Exemplos incluem igual a (==), estritamente igual a (===), diferente de (!=), estritamente diferente de (!==), maior que (>), menor que (<), maior ou igual a (>=) e menor ou igual a (<=).< /p>

Operadores Lógicos:
São usados para combinar expressões lógicas e retornar um valor booleano. Os operadores lógicos incluem o AND lógico (&&), o OR lógico (||) e a negação lógica (!).

Operadores de Concatenação de Strings:
São usados para concatenar strings, usando o operador de adição (+).

Operador Ternário:
É uma forma compacta de escrever uma instrução condicional. O operador ternário (condition ? expression1 : expression2) avalia a condição e retorna a expression1 se a condição for verdadeira e expression2 se for falsa.

Operadores de Acesso a Propriedades:
Permitem acessar as propriedades de um objeto. O operador ponto (.) e o operador de colchetes ([]) são usados para acessar propriedades de objetos.
